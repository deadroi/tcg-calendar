# tcg-calendar  

#1.TCG 대회 일정 수집기  
x api를 이용해서 
전국 카드샵 x에서 데이터를 수집해 대회일정을 정리  
  
텍스트,이미지를 AI로 분석하여 일정을 추출함  
  
  
x에 대회정보를 입력하지 않으면 수집되지 않음  
검색을 허용해야지 수집이됨(검색 허용 OFF, 외부 노출 제한,로그인안하면 못보는글,follow해야만 보이는글은 수집이 힘듬)  
  
기본적으로 갱신은 하루에 한번인데  
아직 개발테스트단계라 2~3번 씩 생신되기도함  
  
#2.TCG 대회 결과 수집기  
  
x api를 이용해서 
전국 카드샵 x에서 데이터를 수집해 대회결과를 정리  
텍스트,이미지를 AI로 분석하여 대회결과를 추출함  
  
x에 대회결과를 입력하지 않으면 수집되지 않음  
검색을 허용해야지 수집이됨(검색 허용 OFF, 외부 노출 제한,로그인안하면 못보는글,follow해야만 보이는글은 수집이 힘듬)  
  

갱신은 1주에 1-2번(월요일은 고정)  
아직 개발테스트단계라 2~3번 씩 생신되기도함  
  
#3. 스위스라운드 시스템  
스위스 라운드는 탈락 없이 모든 참가자가 동일한 라운드 수를 치르는 토너먼트 방식입니다.  
매 라운드마다 비슷한 승점의 참가자끼리 매칭되며, 최종 순위는 아래 타이브레이커 순으로 결정됩니다.  
  
타이브레이커 (동점 시 순위 결정 순서)  
승점 (Match Points) — 승 3점, 무 1점, 패 0점  
OMW% (Opponents' Match-Win %) — 상대들의 매치 승률 평균 (최소 33%)  
GW% (Game-Win %) — 본인의 개별 게임 승률 (최소 33%)  
OGW% (Opponents' Game-Win %) — 상대들의 GW% 평균  
맞대결 결과 — 동률 두 참가자가 서로 대전한 결과  
크롬(Chrome) 또는 엣지(Edge) 브라우저 사용을 권장합니다.  
대회 정보는 사용자 PC의 브라우저 내부(IndexedDB)에만 저장되며, 외부 서버로 전송되지 않습니다.
